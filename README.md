# shopping-trends-dashboard-analysis
This project presents an interactive Shopping Trends Dashboard developed using both Power BI and Looker Studio. The dashboard provides insights into customer purchasing behavior, product performance, revenue distribution, and seasonal sales trends.  

# The objective of this project was to transform raw retail shopping data into meaningful visualizations that support business decision-making and trend analysis.

# Dashboard Tools
-Power BI
Used to create an interactive dashboard with KPI cards, filters, charts, and tables for sales analysis

-Looker Studio (Google Data Studio)
Used to develop a web-based dashboard that presents key shopping trends through interactive visualizations and filtering capabilities

# Dataset Overview
The dataset contains customer shopping information, including:

-Customer ID
-Age
-Gender
-Category
-Item Purchased
-Purchase Amount
-Season
-Review Rating
-Payment Method
-Location
-Subscription Status

# Dashboard Features

KPI Cards
-Total Revenue
-Total Customers

Filters
-Category
-Gender

Visualizations
-Revenue by Category
-Revenue by Gender
-Revenue by Season
-Top Purchased Items

# Key Insights
-Clothing generated the highest revenue among product categories.
-Female customers contributed a larger share of total revenue.
-Winter recorded the highest seasonal sales performance.
-Several products consistently generated higher revenue than others, indicating strong customer demand.
-Revenue trends varied across categories and customer segments.
